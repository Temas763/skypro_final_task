Git

  • git init — создание нового репозитория Git в текущем каталоге.
  • git clone — клонирование репозитория из удаленного хранилища на локальную машину.
  • git status — показывает текущее состояние рабочего каталога и индекса, включая изменения, готовые к коммиту
  • git commit -m"" — сохранение изменений в локальном репозитории с сообщением о коммите.
  • git push — выгрузка сделанных изменений в удаленный репозиторий (например, GitHub).
  • git branch — показывает список всех локальных веток. Используйте git branch для создания новой ветки .
  • git log — просмотр истории коммитов в репозитории.
  • git diff — показывает различия между изменениями в рабочем каталоге и индексов, а также мжду коммитами
  • git remote -v — позволяет посмотреть удалённые репозитории и их URL-адреса.
